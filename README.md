# Language Translation Tool

A Streamlit web app that translates text between 100+ languages using Google Translate,
with text-to-speech playback and translation history.

## Features

- 100+ languages supported including auto-detection of source language
- Text-to-speech output for translated text
- Translation history (last 10 translations)
- Save translation as .txt file
- Character counter (5000 character limit)
- Clean, responsive two-column layout

## How to run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## How it works

Input text is sent to Google Translate via the `deep-translator` library. The translated
result is displayed immediately alongside an audio playback option generated by `gTTS`
(Google Text-to-Speech). Each translation is logged to the session history.

## Dependencies

`streamlit` | `deep-translator` | `gtts`
